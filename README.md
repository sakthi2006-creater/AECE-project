# 📖 Overview

AECE (Autonomous Ethical Cognition Engine) is an intelligent AI system that evaluates user queries using ethical reasoning principles.

Unlike traditional AI systems that only generate responses, AECE analyzes:

- Ethical implications
- Fairness
- Bias detection
- Safety risks
- Transparency
- Explainable AI (XAI)

The goal is to make AI systems more trustworthy, responsible, and human-centric.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sakthi2006-creater/AECE-project.git
```

Move into project

```bash
cd AECE-project
```

Install Dependencies

```bash
npm install
```

Run Server

```bash
npm start
```

or

```bash
node server.js
```
